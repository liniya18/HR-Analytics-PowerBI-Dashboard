# 📊 AI-Powered HR Analytics Dashboard

## 📌 Overview

This project is an interactive HR Analytics Dashboard developed using **Power BI** to analyze employee attrition, salary trends, experience, job roles, education fields, overtime, and marital status.

The dashboard provides business insights that help HR teams identify high-risk employee groups and make data-driven retention decisions.

* Microsoft Excel

## ✨ Features

* Executive Dashboard
* AI Insights Dashboard
* KPI Cards
* Employee Attrition Analysis
* Interactive Filters & Slicers
* Business Recommendations

---

## 📈 Key Metrics

* Total Employees
* High Attrition Count
* Average Experience
* Average Salary

---

## 🔍 Business Insights

* Sales Executives show the highest attrition.
* Employees aged 26–35 have higher turnover.
* Overtime is associated with increased attrition.
* Attrition varies across education fields and marital status.
