# AI-Powered HR Analytics Dashboard

## Overview

This project analyzes employee attrition and workforce trends using Power BI.

## Tools Used

- Power BI
- DAX
- Excel

## Features

- Executive Dashboard
- AI Insights Dashboard
- KPI Cards
- Attrition Analysis
- Interactive Filters


## Key Insights

- Sales Executives show the highest attrition.
- Employees aged 26–35 leave more frequently.
- Overtime is associated with higher employee turnover.
- AI recommendations suggest improving work-life balance and retention strategies.
